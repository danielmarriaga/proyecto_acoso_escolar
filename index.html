El bullying o acoso escolar

¿Que es el bullying?

El bullying es un comportamiento agresivo y repetitivo que se produce en el entorno escolar y que puede causar daño físico y emocional a las víctimas. Este problema afecta a millones de estudiantes en todo el mundo.

  • UNICEF: Causas y Consecuencias
  • StopBullying.gov
  • OPS: Información sobre el Bullying

tipos de bullying

Existen varios tipos de bullying, incluyendo el físico, verbal, social y cibernético. Cada tipo tiene sus propias características y puede afectar a las víctimas de diferentes maneras.

Consecuencias del Bullying

El bullying puede tener graves consecuencias para las víctimas, incluyendo problemas de salud mental, bajo rendimiento académico y, en casos extremos, pensamientos suicidas. Es crucial abordar este problema de manera efectiva.

  • Problemas de salud mental
  • Bajo rendimiento académico
  • Impacto en la autoestima
